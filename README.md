# 링킷
링킷은 익스텐션을 통해 탭 저장 시 AI가 주제를 자동으로 분류하여 탭 정리를 돕는 서비스입니다. 익스텐션을 통한 크롬 탭 모음과 웹 페이지를 통한 링크 정보를 저장할 경우 AI가 자동으로 제목과 태그를 부여해줘서 사용자가 손쉽게 탭을 관리하는걸 도와줍니다.

링킷을 통해 무수히 많은 탭을 열어놓는 개발자들도 쉽게 탭을 관리할 수 있도록 기획되었습니다.

## ✨ 기능 소개
### 주요 기능
- 탭 모음(바구니) / 링크 저장 및 관리
- 링크 검색
- 바구니 공유 및 복사

### 사전 준비

### Required
- pnpm
- Node.js v18 이상

### 실행
```

pnpm install

// Run project
pnpm start:dev
```
